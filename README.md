# AI Resume Scanner

🚀 **AI-powered resume analysis tool** - Upload your resume and get instant feedback!

## Features
- Drag & drop resume upload (PDF, DOC, DOCX, TXT)
- AI-powered scoring (0-100)
- Detailed analysis with strengths & improvements
- ATS compatibility check
- Responsive design
- Instant results

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Modern UI/UX with animations
- Vercel deployment ready
